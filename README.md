# Digital Oscilloscope Using ESP32

## Overview

The Digital Oscilloscope Using ESP32 is a low-cost embedded system project designed to measure and display analog waveforms in real time. The project uses the ESP32 microcontroller and its inbuilt ADC to sample input signals and display them on an OLED screen.

This project is useful for observing voltage levels, waveform shapes, and signal variations from different sensors and electronic circuits. It provides a simple and affordable alternative to traditional oscilloscopes for educational and testing purposes.

## Working Principle

1. The ESP32 reads analog input signals using its inbuilt ADC.
2. The analog values are converted into digital values.
3. The waveform data is processed by the ESP32.
4. The processed waveform is displayed on the OLED screen.
5. User buttons are used to control voltage division, time division, and run/stop operation.
6. The oscilloscope continuously updates the display in real time.
